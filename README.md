# Managing A Support Ticket In ServiceNow

<h2>Objective</h2>
This SOP will outline how to manage a support ticket from start to finish for an employee that is locked out of their account.

<h2>Utilities Used</h2>

- <b> ServiceNow </b>

<h2> Key Steps</h2>

**1. Login into Service Now**

- <b> On the home page under the All tab, select Incidents
- <b> Click on new to create a new ticket

**2. Create a New Ticket**

- <b> Enter in all necessary information regarding the user's issues
- <b> Assign yourself to the ticket and provide a detailed description of the issue
- <b> Click on submit to create the ticket

**3. Resolving the Ticket**

- <b> Document all work in the work notes section
- <b> Communicate with the user and verify problem has been resolved
- <b> Once resolution is confirmed with the user, document ticket has been resolved in resolution notes and click resolve
- <b> Under the All tab click on incidents to confirm ticket shows resolved
